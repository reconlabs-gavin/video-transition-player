# Video Transition Player

유튜브 숏츠 스타일의 부드러운 슬라이드 전환 효과를 가진 동영상 플레이어입니다.

## 주요 기능

- 🎬 여러 동영상을 순차적으로 재생
- 🔄 부드러운 슬라이드 전환 효과 (위/아래 방향)
- ⌨️ 키보드로 간편한 조작
- 🔁 동영상 자동 루프 재생
- 📱 유튜브 숏츠와 유사한 UX

## 설치 방법

```bash
pip install -r requirements.txt
```

## 사용 방법

1. `videos` 폴더에 재생할 동영상 파일을 넣습니다 (.mp4, .avi, .mov, .mkv 지원)

2. 프로그램 실행:
```bash
cd src
python main.py
```

## 조작 방법

- **↓ (아래 화살표)** 또는 **S**: 다음 영상으로 전환
- **↑ (위 화살표)** 또는 **W**: 이전 영상으로 전환
- **Q**: 프로그램 종료

## 프로젝트 구조

```
video-transition-player/
├── src/
│   ├── main.py          # 메인 실행 파일
│   └── video_player.py  # 비디오 플레이어 핵심 로직
├── videos/              # 동영상 파일 저장 폴더
├── requirements.txt     # 필요한 패키지 목록
└── README.md           # 프로젝트 설명서
```

## 기술 스택

- **OpenCV**: 동영상 재생 및 프레임 처리
- **NumPy**: 이미지 배열 연산 및 전환 효과 구현

## 전환 효과

부드러운 슬라이드 전환 효과를 제공하며, Ease-out 애니메이션 커브를 적용하여 자연스러운 움직임을 구현했습니다.

## 향후 개발 계획

- MediaPipe를 활용한 손 제스처 인식 추가
- 다양한 전환 효과 추가 (페이드, 디졸브 등)
- 영상 재생 컨트롤 (일시정지, 빠르기 조절 등)
